# UI Builder 使用帮助 / UI Builder Help Guide ## 简介 / Introduction UI Builder是一款专为iOS/iPadOS开发者设计的UI组件和图标管理工具。它提供了丰富的组件库和系统图标库，帮助开发者快速构建和预览UI界面。 UI Builder is a UI component and icon management tool designed for iOS/iPadOS developers. It provides a rich library of components and system icons to help developers quickly build and preview UI interfaces. ## 功能特性 / Features ### 组件库 / Components Library - 提供常用UI组件的快速访问 - 支持组件预览和属性配置 - 适配iPad和iPhone不同设备类型 - Quick access to common UI components - Support component preview and property configuration - Adapted for both iPad and iPhone devices ### 图标库 / Icons Library - 集成iOS/iPadOS系统图标 - 支持按类别浏览图标 - 提供图标搜索功能 - 支持不同尺寸预览 - Integrated iOS/iPadOS system icons - Browse icons by categories - Icon search functionality - Preview icons in different sizes ## 使用指南 / Usage Guide ### 组件库使用 / Using Components Library 1. 点击底部标签栏的"组件库"进入 2. 浏览并选择所需组件 3. 查看组件详情和使用方法 1. Tap "Components" in the bottom tab bar 2. Browse and select desired components 3. View component details and usage instructions ### 图标库使用 / Using Icons Library 1. 点击底部标签栏的"图标库"进入 2. 使用顶部搜索栏查找图标 3. 点击类别标签筛选图标 4. 选择图标查看详细信息 1. Tap "Icons" in the bottom tab bar 2. Use the top search bar to find icons 3. Filter icons by category tags 4. Select an icon to view details ## 设备适配 / Device Compatibility ### iPad版本特性 / iPad Version Features - 更大的标签栏设计 - 优化的界面布局 - 支持分屏多任务 - Larger tab bar design - Optimized interface layout - Support for split-screen multitasking ### iPhone版本特性 / iPhone Version Features - 紧凑的界面设计 - 优化的单手操作体验 - Compact interface design - Optimized one-handed operation experience ## 技术支持 / Technical Support 如有任何问题或建议，请通过以下方式联系我们： - 电子邮件：ningappsupport@163.com For any questions or suggestions, please contact us through: - Email: ningappsupport@163.com
